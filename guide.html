🚀 Getting Started

Quick Setup (5 minutes)

  1. Open Scout Tracker
    Go to lamangamatt.github.io/scout-tracker on your phone, tablet, or computer.
  2. Create a Team Code
    Enter a unique code for your team (e.g., bingham-varsity). This keeps your data separate from other teams. All coaches on your team will use this same code.
  3. Add to Home Screen
    For instant access on game day, add it to your home screen like a regular app.
  4. Set Up Your Roster
    Go to Team Admin → add your players (name, number, positions).
  5. Share with Assistant Coaches
    Give them your team code — they'll enter it when they first open the app.
📱 Add to Home Screen

iPhone/iPad: Tap the Share button (box with arrow) → "Add to Home Screen"
Android: Tap the menu (three dots) → "Add to Home Screen" or "Install App"

This gives you an app icon and lets Scout Tracker work offline!

🔑 Understanding Team Codes

Team codes are the key to how Scout Tracker works. They keep each team's data completely separate and allow multiple people to collaborate.

How Team Codes Work

  • One code per team — Everyone tracking the same team uses the same code
  • Data syncs in real-time — When the scorekeeper logs a pitch, the coach sees it instantly
  • Multiple teams — Use different codes for JV, Varsity, etc.
  • Case insensitive — BINGHAM and bingham are the same

Choosing a Good Team Code

✅ Good Codes ❌ Avoid
bingham-varsity-2026 team1 (too generic)
miners-jv my team (no spaces)
stoker-soph baseball (too common)

Sharing Your Team Code

From the Team Admin page, you can share your code via:

  • Copy button — Copies to clipboard for texting
  • Share button — Uses your phone's share menu
  • QR Code — Show it for others to scan
⚠️ Keep Your Code Private
Anyone with your team code can view and edit your data. Only share it with coaches and trusted helpers.

Managing Multiple Teams

If you coach multiple teams (JV, Varsity, etc.), use a different code for each:

  • bingham-varsity — Varsity team data
  • bingham-jv — JV team data
  • bingham-soph — Sophomore team data

Switch between teams using the "Switch Team" button in any page header.

📝 Data Entry (Game Tracking)

This is where you'll spend most of your time during games. It's designed for quick, one-handed operation from the dugout or stands.

Recording a Pitch

  1. Select the Pitcher & Batter
    Use the dropdowns at the top. The app remembers your selection until you change it.
  2. Tap Pitch Type
    FB (Fastball), CB (Curveball), SL (Slider), CH (Changeup), CT (Cutter), or SI (Sinker).
  3. Tap Location
    Use the 9-zone grid showing the strike zone from the catcher's perspective.
  4. Tap Result
    Ball, Called Strike, Swinging Strike, Foul, In Play (specify hit type or out).

The pitch is logged instantly. The count, pitch total, and all stats update automatically.

Pitch Count Alerts

The colored bar at the top shows the current pitcher's pitch count with visual alerts:

✅ Green Zone Under warning threshold — pitcher is fresh
⚠️ Yellow Zone At warning threshold (default: 70) — start warming someone up
🛑 Red Zone At or over limit (default: 85) — consider removing pitcher
💡 Customize Limits
Set your league's pitch count limits in Team Admin → Pitch Count Settings. Different leagues have different rules!

Game Management

  • Next Batter — Tap after a plate appearance ends to advance
  • Change Pitcher — Select new pitcher from the dropdown
  • Undo — Remove the last logged pitch if you made a mistake
  • End Game — Saves the game to history and resets for next game

In-Game Scouting

If you have scouting data on the opponent, a "📋 View Scouting" button appears. Tap it to see:

  • Current batter's tendencies and weaknesses
  • How to pitch to them
  • Historical results against your team

📊 Coach View (Live Dashboard)

The Coach View is a real-time dashboard that updates automatically as pitches are logged. Perfect for:

  • A tablet in the dugout
  • An assistant coach watching from the stands
  • A parent helping track the game

What You'll See

🔢

Live Count

Balls, strikes, outs, inning — always current

📈

Pitch Distribution

What percentage of each pitch type has been thrown

🎯

Location Heatmap

Visual map of where pitches are landing

💪

Pitcher Stats

Strike %, total pitches, K's, walks

Real-Time Sync

The Coach View uses live Firebase sync. When someone logs a pitch on Data Entry, everyone with Coach View open sees it within 1-2 seconds — no refresh needed.

The sync indicator shows:

  • 🟢 Live — Connected to Firebase, real-time updates active
  • 🟡 Local — Using local data (Firebase unavailable)
  • 🔴 Offline — No connection
✨ Pro Setup
Have one person on Data Entry logging pitches, and the coach watching Coach View on a second device. The coach sees analytics without having to enter data.

🎯 Bullpen Tracker

Track bullpen sessions, practice, and warm-ups separately from games. Great for:

  • Monitoring between-game workload
  • Tracking bullpen sessions at practice
  • Pre-game warm-up counts

Starting a Session

  1. Select the pitcher from your roster
  2. Log pitches — pitch type, location, and quality rating
  3. Add notes — "Working on curve", "Arm felt tired", etc.
  4. End session — Saves to pitcher's history

Session Data

  • Total pitches thrown
  • Pitch type breakdown
  • Command rating (if entered)
  • Notes for each session
📊 Workload Monitoring
Bullpen sessions count toward weekly arm load. Track sessions to avoid overworking pitchers during practice weeks.

📋 Pre-Game Scouting Reports

Before a game, pull up everything you know about the opponent in one printable report.

Generating a Report

  1. Go to Scouting Report
  2. Select the opponent team from the dropdown
  3. View the full report with all players and notes

Report Contents

For Each Batter:

  • Name, number, batting side
  • Tendencies (pull, ball flight)
  • Weaknesses highlighted in red
  • Strengths highlighted in green
  • "How to Pitch Him" — Auto-generated approach based on weaknesses
  • Past results against your team (K, BB, 1B, 2B, OUT, etc.)

For Each Pitcher:

  • Name, number, throwing side
  • Pitch repertoire with velocity
  • Tendencies by count
  • "How to Beat Him" — Approach notes
  • Fatigue indicators

Sharing the Report

  • 🖨️ Print — Post it in the dugout
  • 📤 Share — Send via text or email

💡 Tips & Frequently Asked Questions

Best Practices

📱 Dedicated Device
Have one person responsible for Data Entry on a dedicated phone or tablet. Fewer hands = fewer mistakes.
🔋 Battery & Screen
Turn up screen brightness, enable "Stay Awake" if available, and bring a battery pack for doubleheaders.
📶 Bad Signal?
Scout Tracker works offline! Data saves locally and syncs when you get connection back.
🔄 Regular Backups
Export a backup from Admin after each tournament. If someone accidentally clears data, you can restore it.

FAQ

Q: What if multiple people are entering data at the same time?

A: The last entry wins. It's best to have one designated person on Data Entry to avoid conflicts.

Q: Can I use this without internet?

A: Yes! Data saves to your device locally. When you get internet back, it syncs to Firebase so other devices can see it.

Q: I lost my team code. How do I find it?

A: Check with another coach who has access. If no one remembers, you may need to start fresh with a new code.

Q: Can I track multiple teams?

A: Yes! Use a different team code for each team (e.g., bingham-varsity, bingham-jv). Switch between them using the "Switch Team" button.

Q: How do I delete a pitch I logged wrong?

A: Use the Undo button on the Data Entry page to remove the last pitch.

Q: Is my data private?

A: Your data is only accessible to people who know your team code. Don't share your code publicly.

Q: Does this cost anything?

A: Nope! Scout Tracker is completely free.
