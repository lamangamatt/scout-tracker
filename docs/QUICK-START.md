# Scout Tracker Quick Start

**Get tracking in 5 minutes or less!** ⚾

---

## Step 1: Open the App

Go to: **[lamangamatt.github.io/scout-tracker](https://lamangamatt.github.io/scout-tracker/)**

Works on any device with a browser - phone, tablet, or computer.

---

## Step 2: Set Your Team Code

1. Enter a unique code (e.g., your team name: `miners2025`)
2. Click **Set**
3. Share this code with anyone who needs to track or view

> **Pro Tip:** Same code = synced data. Give it to your assistant coaches!

---

## Step 3: Add Your Teams

Before your first game:

1. Tap **Data Entry**
2. Click the **⚙️** button
3. Enter your team name
4. Add your roster (one player per line):
   ```
   1 John Smith
   2 Mike Johnson
   24 Chris Davis
   ```
5. Click **Save Team**

Repeat for common opponents.

---

## Step 4: Track a Game

**Setup:**
- Select batting team (opponent when you're pitching)
- Select pitching team (your team when you're pitching)
- Choose current batter and pitcher

**For each pitch:**
1. Tap the **pitch type** (FB, CB, SL, etc.)
2. Tap the **location** on the grid
3. Tap the **result** (Ball, Called, Swing, Foul, In Play)

That's it! The app tracks everything else.

---

## Step 5: Use Coach View (Optional)

On a second device:
1. Go to Scout Tracker with the same team code
2. Tap **Coach View**
3. Watch live pitch predictions and insights while someone else enters data

---

## Key Buttons

| Button | What it does |
|--------|--------------|
| **Next AB** | New at-bat (resets count) |
| **New Inn** | New inning |
| **Undo** | Removes last pitch |
| **🎤 Tap to Speak** | Voice entry ("fastball inside called") |

---

## Pitch Count Colors

- 🟢 **Green** = Safe
- 🟡 **Yellow** = Approaching limit
- 🔴 **Red** = At/over limit

Set your limits in **Team Admin** → **Pitch Count Limits**.

---

## Add to Home Screen

**iPhone:** Safari → Share → Add to Home Screen

**Android:** Chrome → Menu → Add to Home Screen

Now it works like an app!

---
